Existência e Unidade do Polinômio Interpolador

Basta só montar sistema linear:

pn(xk)=f(xk)p_n(x_k) = f(x_k)pn​(xk​)=f(xk​) com pnp_npn​ polinômio de grau n, para cada k de 0 a n.

Temos n+1 equações e n+1 incógnitas (a0a_0a0​, ..., ana_nan​)

Assim, temos uma matriz A, igual a de Vandermonde. (1 x0x_0x0​ x02x_0 ^2x02​ x02x_0^2x02​ ... x0nx_0^nx0n​ seria a primeira linha, e etc)

Note que, resolver por Gauss já resolveria esse sistema, para achar os a0a_0a0​, ..., ana_nan​.
