Raises são valores de x (real ou complexo) tal que f(x)=0

Elas seram obtidas com uma determinada precisão, temos um erro tolerável, a pretenção não é obter valores exatos.

Ex.: f(x) = ax^2 + bx + c = 0 quando x = (-b±b2−4a\pm\sqrt{b^2 -4a}±b2−4a​)/2a

Equação Transcendentais não podemos isolar o x.

Ex.: cosx = x

Método Gráfico

É para ter uma ideia através do esboço, ai com esses valores podemos usar esses valores com precisão maior. Queremos analisar o intervalo onde troca de sinal. Desde que continua na região.

Ex.: f(x) = cos(x) - x => g(x) - h(x) = 0 => x = cos(x), esboça os dois gráficos e pega a interceção das duas curvas.

Método da Bisseção

A função tem que ser continua em [a,b] e tal que f(a)f(b)f(a)f(b)f(a)f(b) < 0 (tem uma raiz dentro). Vamos supor que existe uma raiz ϵ\epsilonϵ de f(x).

Esse método tem como objetivo reduzir a aimplitude do intervalo até que contém a raiz até ase atingir a precisão deseja (b-a) < E\EpsilonE, fazendo sucessivas divisões de [a,b] ao meio.

Algoritmo:

  • Lembre a exigência
  • f((a+b)/2) = f(c)
  • Se f(c) = 0, temos a raiz
  • Se f(c)f(a) < 0, a raiz está entre a e c, então b = c
  • Se não está entre c e b, então a = c
