Critério de Convergência - Matriz Diagonal Dominante

Se somar em modulo os coeficientes fora da diagonal (na mesma linha), em módulo, e o elemento da diagonal domina essa soma. Temos que a matriz é diagonal dominante. Assim será convergente, caso não seja, não poderemos garantir nada.

Se for Diagonal Dominante, então a matriz sempre terá essa propriedade.

O critério, então, pode ser representado criando constantes αi\alpha_iαi​:

αi=(∑j=1≠in∣aij∣)/∣aii∣\alpha_i = \left(\sum_{j=1\neq i}^{n}|a_{ij}|\right) / |a_{ii}|αi​=⎝⎛​j=1=i∑n​∣aij​∣⎠⎞​/∣aii​∣, com i = 1, 2, ..., n

Se α=max⁡αi<1\alpha = \max\alpha _i < 1α=maxαi​<1, então ambos métodos convergem para qualquer aproximação inicial.

A mais rápida é a de Gauss-Seidel.

Formulação Matricial dos Métodos Iterativos Clássicos

Ax=bAx = bAx=b
Decompondo a matriz A em:

A=D−E−FA = D - E - FA=D−E−Fonde:

  • D é a matriz diagonal com elementos da diagonal não nulo.
  • E a matriz triangular estritamente inferior
  • F a matriz triangular estritamente superior.

Fazendo Ax = b, substituindo A:

Dx=(E+F)x+bDx = (E+F)x + bDx=(E+F)x+b
x=D−1(E+F)x+D−1bx = D^{-1}(E+F)x + D^{-1}bx=D−1(E+F)x+D−1b

Assim, a formulação iterativa:

x(k+1)=(L+U)x(k)+D−1bx^{(k+1)} = (L + U)x^{(k)} + D^{-1}bx(k+1)=(L+U)x(k)+D−1b

x(k+1)=Jx(k)+dx^{(k+1)} = Jx^{(k)} + dx(k+1)=Jx(k)+d onde J=L+U=D−1(E+F)J = L + U = D^{-1}(E+F)J=L+U=D−1(E+F)

Essa é a formulação Matricial do Método Iterativo de Gauss-Jacobi. Agora já Gauss-Seldel:

Ax=bAx = bAx=b

(D−E−F)x=b(D-E-F)x = b(D−E−F)x=b

(D−E)x=Fx+b(D-E)x = Fx + b(D−E)x=Fx+b

D−1(D−E)x=D−1Fx+D−1bD^{-1}(D - E)x = D^{-1}Fx + D^{-1}bD−1(D−E)x=D−1Fx+D−1b

(I−L)x=Ux+D−1b(I - L)x = Ux + D^{-1}b(I−L)x=Ux+D−1b

x=(I−L)−1Ux+(I−L)−1D−1bx = (I - L)^{-1}Ux + (I - L)^{-1}D^{-1}bx=(I−L)−1Ux+(I−L)−1D−1b

Fazendo a forma iterativa:

x(k+1)=(I−L)−1Ux(k)+(I−L)−1D−1bx^{(k+1)} = (I - L)^{-1}Ux^{(k)} + (I - L)^{-1}D^{-1}bx(k+1)=(I−L)−1Ux(k)+(I−L)−1D−1b
